# RandomStatus
ゲーム内の時間帯ごとに、全ヒロインのステータス（性格、関係性、H度など）を自動でランダム変動させるプラグインです。 常に新鮮なキャラクターを楽しめます。 変化は恒常的なものです。 途中から機能をオフにすれば、変動後の性格に固定してゲームを楽しむといったこともできます。  機能: 「関係性」「H度」「性格」「訪問願望（KKSのみ）」の自動変動。 H度に応じて「淫乱」状態を自動付与。 H度変動に対して特定の性格にだけ倍率を適用。 「行為拒否解除」「レズ・オナニー願望」の一括最大化。  カスタマイズ: 「Plugin settings」より各項目のON/OFFおよび変動比率のスライダー調整が可能。 エクステンション版などへ対応するため全性格数設定。（エクステンション版は44かな？）
